Signature.io

Signature.io is the API Platform for Signing Documents.

Signing Up

To create your account do the following, replacing the filler information with your information. This will return an API key for your use. See documentation for more details.

curl -X POST https://www.signature.io/api/v0/people.json \ -d "first_name=YOUR_FIRST_NAME" \ -d "email=YOUR_EMAIL" \ -d "password=YOUR_PASSWORD"
